# encuesta-preinscripcion-reactjs

> A ReactJS project

## Installation

With npm:

``` bash
# install dependencies
npm install
```

## Build Setup

``` bash

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

```

## Tests

``` bash
# run all tests
npm test
```

## Run

``` bash
# run
npm run
```
